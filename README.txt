ÁLBUM A JORNADA DE VALENTINA — V6

Correções desta versão:
- Monograma V da capa ajustado para não cortar.
- Game oficial Match 3 integrado no botão "Jogar para ganhar figurinhas".
- Áudios do game colocados na pasta game/audio.
- Música de fundo do álbum mantida em audio/bg-music.mp3.
- Bomba do game usa audio/bomba.mp3.
- Figurinhas aparecem dentro da tela do game em game/figurinhas.
- 1 pacote grátis na primeira abertura do álbum.
- Cada fase concluída no game libera 1 pacote com 4 figurinhas sem repetidas.
- Total de 8 fases para completar 36 figurinhas.

Como abrir:
1. Extraia a pasta.
2. Abra index.html.
3. Clique em Música para ativar o áudio do álbum.
4. Clique em Jogar para ganhar figurinhas.

Estrutura:
- index.html = álbum principal
- audio/bg-music.mp3 = música do álbum
- figurinhas/01.webp até 36.webp = figurinhas do álbum
- game/index.html = game Match 3
- game/audio/ = áudios do jogo
- game/figurinhas/001.webp até 036.webp = figurinhas usadas dentro do jogo
